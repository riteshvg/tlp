I want to create a segment that gives me mobile visitors by country like India, US etc. How can I make it in CJA?

Creating Mobile Visitors by Country Segments in CJA

Great use case! Rather than building one large complex segment, CJA supports segment stacking — which is the most efficient approach for this scenario.

Recommended Approach: Stacked Segments

Instead of building 50+ individual segments (one per country + device combination), build a small set of reusable segments and stack them to get your desired combinations.

Prerequisites

  • Access to Customer Journey Analytics Workspace
  • The relevant Device Type and Country dimensions must be available in your data view

Step 1: Create a "Mobile" Segment

1. Go to Components > Segments in CJA

2. Click + Create

3. In the Segment Builder, name it `Mobile Visitors`

4. Set the container to Person (or Session/Event depending on your use case)

5. Add a rule: Device Type → equals → `Mobile Phone`

6. Click Save

Step 2: Create Country Segments

Repeat the same process for each country you need:

For India:

1. Create a new segment named `Country - India`

2. Add rule: Country → equals → `India`

3. Click Save

For US:

1. Create a new segment named `Country - US`

2. Add rule: Country → equals → `United States`

3. Click Save

Repeat for any additional countries you need.

Step 3: Stack the Segments in Workspace

Once your base segments are created, stack them in Analysis Workspace:

1. Open your Analysis Workspace project

2. Drag the `Mobile Visitors` segment to the segment drop zone at the top of your panel

3. Drag `Country - India` on top of the existing segment — CJA will automatically stack them

4. The combined result gives you Mobile Visitors in India

To get Mobile Visitors in the US, simply swap out `Country - India` for `Country - US`.

Why This Approach Works Better

| Approach | Number of Segments Needed |

|---|---|

| One segment per combination | 50+ segments (one per country) |

| Stacked segments (recommended) | ~52 segments reused flexibly |

  • ✅ Reusable — your `Mobile Visitors` segment can be reused across any country
  • ✅ Easier to maintain — update one segment and it applies everywhere
  • ✅ Less complexity — no need to rebuild logic repeatedly

Optional: Include Tablet Visitors

If you also want Tablet visitors, simply create a separate `Tablet Visitors` segment and stack it with any country segment the same way. You can also combine both mobile and tablet into a single segment using an OR condition:

  • Device Type → equals → `Mobile Phone`
  • OR Device Type → equals → `Tablet`

This gives you a single `Mobile & Tablet Visitors` segment to stack with any country.
