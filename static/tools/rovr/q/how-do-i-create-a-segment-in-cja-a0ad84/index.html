How do I create a segment in CJA?

Creating Segments in Customer Journey Analytics

Segments in CJA let you identify subsets of persons, sessions, or events based on attributes, interactions, custom variables, and other criteria.

Prerequisites

  • By default, only administrators can create segments.
  • Admins can grant the [!UICONTROL Segment Creation] permission to users via [!UICONTROL Reporting Tools] in [!UICONTROL Edit permissions for CJA Workspace Access] in the Admin Console.

Ways to Create a Segment

There are six ways to create a segment in CJA:

A — From the Main Interface (Segment Manager)

1. Select [!UICONTROL Components] from the top navigation.

2. Select [!UICONTROL Segments].

3. Select [!UICONTROL Add] (the + icon) from the Segment Manager.

B — From the Components Panel in a Workspace Project

1. Open a Workspace project.

2. In the left components panel, select the + icon next to the Segments section.

C — From a Visualization Context Menu

1. In a Workspace project, right-click within a visualization.

2. Select [!UICONTROL Create segment from selection].

D — From the Workspace Menu

1. In a Workspace project, select [!UICONTROL Components] from the top menu.

2. Select [!UICONTROL Create segment].

E — Using a Keyboard Shortcut

  • macOS: `shift + cmd + e`
  • Windows: `shift + ctrl + e`

F — From a Drop Zone

  • Drag any component into a Drop a segment here drop zone in a panel. This creates a project-only segment.

Segment Creation Options

Once you initiate segment creation, you have two builder options:

Full Segment Builder

Used for complex, reusable segments that apply across all projects. Supports:

  • Person, Session, and Event containers
  • Sequential segmentation using the THEN operator
  • Nesting and combining multiple conditions

Quick Segment Builder

A simplified, in-project option for fast segmentation based on simple criteria. To use it:

1. Select the + icon at the top of a panel.

2. Build your condition(s) directly in the Quick Segment Builder interface.

3. Results are applied immediately to the panel.

> Quick segments are panel-scoped and can be created by any Workspace user — no admin permission required.

Segment Container Terminology (vs. Adobe Analytics)

| Adobe Analytics | Customer Journey Analytics |

|---|---|

| Hit container | Event container |

| Visit container | Session container |

| Visitor container | Person container |
