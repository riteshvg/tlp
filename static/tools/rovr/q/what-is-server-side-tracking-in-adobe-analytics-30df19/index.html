What is Server Side Tracking in Adobe Analytics?

I don't have specific documentation on "Server Side Tracking" in the retrieved context. However, I do have information on a related Adobe Analytics feature that might address what you're looking for.

Server-Side Forwarding is a feature in Adobe Analytics that allows data to be shared between Analytics and other Adobe Experience Cloud solutions in real time. Here's what it does:

Key Benefits of Server-Side Forwarding

  • Reduces page calls: Server-side forwarding eliminates the need for additional data collection calls from the page. For example, Audience Manager customers no longer need separate DIL (Data Integration Library) code, which improves page load times and user experience.
  • Enables data sharing: It allows Analytics to push data to other Experience Cloud solutions, and those solutions can push data back to Analytics during the data collection process.
  • Best practice implementation: Server-side forwarding is the recommended method for implementing data collection and transfer between Adobe solutions, especially for new customers.
  • GDPR compliance support: Organizations can restrict pre-consent data from being forwarded to other solutions until proper consent is received.

How It Works

When server-side forwarding is enabled, data collected by Analytics is automatically forwarded to other CX Enterprise solutions on the server side, rather than requiring client-side code to make additional requests.
