What is Agent Orchestrator?

Agent Orchestrator Overview

Agent Orchestrator is the agentic layer in Adobe Experience Platform that powers customer experience orchestration with purpose-built agents, an advanced reasoning engine, and a powerful conversational interface.

Key Capabilities

Agent Orchestrator allows you to:

  • Access specialized agents — Leverage purpose-built expert Adobe Experience Platform agents that handle complex decision-making and problem-solving tasks
  • Work conversationally — Complete complex end-to-end workflows through a natural language conversational interface without needing to know which specific agents are working behind the scenes
  • Maintain context — Remember your conversation history, enabling you to build on previous questions naturally without repeating context
  • Combine insights — Integrate information from multiple agents to present unified, clear responses
  • Understand reasoning — Explore the Agent Orchestrator reasoning panel within your conversation to see the step-by-step thinking process

Core Components

Agent Orchestrator consists of several key components:

1. AI Assistant conversational interface — The primary user-facing interface for natural language interaction

2. Reasoning engine — Powers decision-making and planning capabilities

3. Specialized Adobe Experience Platform agents — Purpose-built agents that execute specific tasks

4. Knowledge base — Provides access to relevant information needed to fulfill requests

Where You Can Use It

Agent Orchestrator functionality is accessible in multiple CX Enterprise applications, including:

  • Adobe Journey Optimizer
  • Real-Time CDP
  • Adobe Experience Platform

The system automatically orchestrates the right agents based on your needs, creates step-by-step plans, and adjusts its approach based on your feedback.
