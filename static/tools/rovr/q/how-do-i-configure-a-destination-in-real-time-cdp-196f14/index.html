Configuring a Destination in Real-Time CDP

Based on the available documentation, here is an overview of how destination configuration works in Real-Time CDP.

What is a Destination?

Destinations are pre-built integrations with partner platforms that allow you to activate audience and profile data from Real-Time CDP. They are accessible via the Destinations Catalog, which displays available destination integration cards.

General Configuration Steps

Prerequisites

  • You must have the appropriate Adobe product licenses (e.g., an Audience Manager license is required to share audiences with Adobe Analytics)
  • Ensure your data schemas and audiences are already set up in AEP

Step-by-Step Configuration

1. Navigate to the Destinations Catalog in the Real-Time CDP UI and browse or search for your desired destination

2. Select your destination card from the catalog

3. Authenticate to the destination by filling in the required credential fields (e.g., username and password) and selecting Connect to destination

4. Fill in destination details, including:

  • Name – A recognizable name for this destination
  • Description – Helps identify the destination in the future
  • Company/Account-specific fields – Varies depending on the destination

5. Configure grouping and governance policies if applicable

6. Enable alerts to receive notifications about data flows to the destination

7. Select the Create button to finalize the connection

Important Notes on Specific Destinations

  • Adobe Analytics: Audiences sent via the Experience Cloud Audiences destination do not appear automatically — you must implement the Experience Cloud Identity Service for Analytics and Audience Manager first
  • Adobe Target, Adobe Advertising, Marketo, Adobe Campaign: Adobe recommends using their dedicated destination cards rather than the generic Experience Cloud Audiences destination card for these solutions
  • Sharing audiences with Adobe Analytics requires an Audience Manager license, but sharing with Adobe Target, Adobe Advertising, Marketo, and other solutions does not

Available Destination Types

Real-Time CDP supports more than 60 destinations natively, including:

  • Advertising destinations (e.g., Google Customer Match, Amazon Ads, The Trade Desk)
  • Social destinations (e.g., Facebook, LinkedIn, TikTok, Snapchat, Pinterest)
  • Email marketing (e.g., Adobe Campaign)
  • Personalization (e.g., Adobe Target)
