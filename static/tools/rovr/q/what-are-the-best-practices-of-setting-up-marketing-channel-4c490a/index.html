Best Practices for Setting Up Marketing Channels in Customer Journey Analytics

Key Differences to Understand First

Marketing channels in Customer Journey Analytics operate differently than in traditional Adobe Analytics. Here are the critical differences:

Report-Time Processing

Customer Journey Analytics processes marketing channels at report time (not at data collection time), which means:

  • Sessions are determined at query time rather than collection time
  • This enables non-destructive, retroactive channel configuration

"Is First Page of Visit" Limitation

Any marketing channel rule using the "Is First Page of Visit" criteria will be ignored in Platform data. If you're migrating from Adobe Analytics, you should re-evaluate these rules and use alternative approaches to achieve your goals. Other criteria in the same rule will still apply.

Recommended Setup Approaches

1. Use Derived Fields (Recommended)

CJA provides a non-destructive, retroactive way to define marketing channels using Derived Fields with the Marketing Channel function template within your Data Views. This approach allows you to:

  • Modify channel definitions without losing original data
  • Retroactively apply changes to historical data
  • Test and refine your logic over time

2. Carry Over Rules from Adobe Analytics

If you're using the Analytics source connector, you can carry over marketing channel processing rules from your Adobe Analytics report suite. However, be aware of the "Is First Page of Visit" limitation mentioned above.

3. Define Custom Channel Groupings

Unlike Google Analytics, CJA has no built-in channel groupings. You must explicitly define them based on your business needs using:

  • Derived fields with custom logic
  • Referrer domain and type dimensions
  • UTM parameters (manually mapped to XDM fields)
  • Tracking codes and other custom dimensions

Key Capabilities to Leverage

Once set up, marketing channels in CJA enable you to:

  • Understand channel effectiveness: Use the Channel Overview Report template to see which marketing channels are most effective at driving visitors to your site
  • Analyze first-touch attribution: Identify which channels drive initial traffic and optimize marketing investments accordingly
  • Review channel details: Examine specific details about how visitors matched with channels (e.g., which search engine or keyword for paid search)
  • Implement advanced attribution: Support complex, multi-touch attribution models across all connected data sources

Additional Resources

Adobe has published updated best practices specifically designed to help you succeed with marketing channels in CJA and prepare for your transition from traditional Analytics. These recommendations emphasize leveraging CJA's advanced attribution features and report-time processing capabilities.
