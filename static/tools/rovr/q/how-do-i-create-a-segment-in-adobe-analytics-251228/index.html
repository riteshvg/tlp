How do I create a segment in Adobe Analytics?

Creating a Segment in Adobe Analytics

Adobe Analytics provides several ways to create segments, depending on your needs and where you're working.

Prerequisites

  • Ensure you have the appropriate segment creation rights for your role in your organization.
  • Segments can be scoped to a single Workspace project or shared across all projects.

Ways to Create a Segment

There are six entry points for creating a segment:

| Option | How |

|--------|-----|

| A – Segment Manager | In the main interface, select Components > Segments, then click ![Add] Add in the Segment Manager. |

| B – Workspace Components Panel | In a Workspace project, find the Segments section in the left Components panel and click the + (Add) icon. |

| C – Visualization Context Menu | In a Workspace project, right-click within a visualization and select Create segment from selection. |

| D – Workspace Menu Bar | In a Workspace project, select Components from the top menu, then choose Create segment. |

| E – Keyboard Shortcut | In a Workspace project, press Shift + Cmd + E (macOS) or Shift + Ctrl + E (Windows). |

| F – Drop Zone | Drag and drop into any Drop a segment here zone — this creates a Project-only segment. |

Building the Segment (Segment Builder)

Once you open the segment creation dialog, the Segment Builder appears:

1. Name your segment — Give it a clear, descriptive title.

2. Add a definition — Drag dimensions, metrics, or events into the definition area to set your segment criteria.

3. Set the container level — Choose between Hit, Visit, or Visitor scope depending on how broadly you want the segment to apply.

4. Combine conditions — Use AND / OR / THEN logic to refine your criteria.

5. Preview the data — The builder shows a data preview based on the last 90 days so you can validate your segment before saving.

6. Check product compatibility — A small number of segment criteria are not compatible with all Adobe Analytics tools; the builder indicates which tools are compatible.

7. Save — Click Save to make the segment available across projects, or save it as a project-only segment if that is your intent.

Quick Segments (Simplified Option)

If you're inside a Workspace project and need a fast, simple segment, you can use Quick Segment creation — a streamlined version of the Segment Builder that lets you define basic conditions without leaving your analysis flow.

Performance Tip

If your segment requires many OR conditions on the same variable (e.g., Page equals X OR Page equals Y OR Page equals Z for hundreds of values), consider using Classifications instead:

1. Create a classification for the variable.

2. Download and fill in the classification template, assigning the same value to all dimension items you want to include.

3. Import the classification back into Adobe Analytics.

4. Once processed, build a segment using the single classified value.

This dramatically improves segment performance and makes future edits much easier.

Moving Segments to Customer Journey Analytics

If you plan to use your segments in Customer Journey Analytics, note that segments are called filters in CJA. The logic can be re-created using CJA's Filter Builder.
