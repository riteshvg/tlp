What are the different ways to ingest data into Adobe Experience Platform?

Data Ingestion Methods in Adobe Experience Platform

Adobe Experience Platform supports ingesting data from external sources and allows you to structure, label, and enhance that data using Experience Platform services. You can bring in data from a wide variety of source types.

Supported Source Categories

AEP can ingest data from:

  • Adobe applications (e.g., Adobe Analytics, Marketo Engage, Adobe Audience Manager)
  • Cloud-based storage (e.g., Amazon S3, Azure Blob, Google Cloud Storage)
  • Third-party software platforms
  • CRM systems (e.g., Salesforce, Microsoft Dynamics)

Ingestion Methods

1. Batch Ingestion

Ingest data in bulk at scheduled intervals using the Batch APIs. This is ideal for large, periodic data loads.

2. Streaming Ingestion

Send data in real time using the Streaming APIs, enabling continuous data flow into the platform with low latency.

3. Adobe-Built Source Connectors

Use pre-built connectors for Adobe applications and popular third-party systems. These connectors allow you to:

  • Authenticate and connect to external storage systems and CRM services
  • Set times for ingestion runs
  • Manage data ingestion throughput

4. Data Integration Partners

Leverage certified third-party integration partners to connect and pipe data into AEP.

5. Adobe Experience Platform UI

Set up source connections interactively through the Sources section of the AEP UI — no code required.

6. RESTful API

Configure and manage source connections programmatically via Experience Platform's RESTful APIs, making it easy to integrate with enterprise solutions.

7. Web SDK / Edge Network (Client-Side Collection)

Collect client-side customer experience data and send it to the Adobe Experience Platform Edge Network, where it can be enriched, transformed, and distributed to Adobe or non-Adobe destinations.

Accelerators: Data Ingestion Templates

AEP also offers data ingestion templates to speed up the process for data architects and engineers. These templates provide:

  • Standard templates and automation tools
  • Automated schema and dataset creation
  • Mapping rules configuration

> Currently available for Marketo Engage, Salesforce, and Microsoft Dynamics sources.

By combining these methods, AEP supports any type and any latency of data ingestion — from real-time streaming to scheduled batch loads.
