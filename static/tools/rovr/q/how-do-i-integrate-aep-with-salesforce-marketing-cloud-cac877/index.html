How do I integrate AEP with Salesforce Marketing Cloud?

What the documentation confirms

The CONTEXT confirms that Adobe Experience Platform (AEP) offers multiple distinct integrations with Salesforce Marketing Cloud products:

1. Salesforce Marketing Cloud Account Engagement (V2) destination — exports AEP profile data to Salesforce's B2B marketing automation platform using batch processing and the Salesforce Import API v5.

2. (API) Salesforce Marketing Cloud destination — a streaming API-based connection ideal for B2C use cases, using the update contacts API to add and update contact data.

3. Salesforce Marketing Cloud sources — connectors to bring Salesforce Marketing Cloud data into AEP (including a newer V2 source).

What it does NOT cover

The documentation does not contain step-by-step integration procedures, configuration details, authentication workflows, or field mapping instructions for any of these connections.
