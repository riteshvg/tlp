How do I integrate AEP with Salesforce Marketing Cloud?

Integrating Adobe Experience Platform (AEP) with Salesforce Marketing Cloud

There are multiple integration patterns available depending on your direction of data flow and use case. Here's a comprehensive overview:

Integration Options Overview

| Direction | Method | Best For |

|---|---|---|

| AEP → SFMC | Destination (Files) | Batch audience export via file |

| AEP → SFMC | Destination (API) | Streaming/B2C contact updates |

| AEP → SFMC Account Engagement | Destination (V2) | B2B lead management (Pardot) |

| SFMC → AEP | Source Connector | Ingesting SFMC data into AEP |

Option 1: Export AEP Audiences TO Salesforce Marketing Cloud

A. File-Based Destination (Batch)

The [!DNL (Files) Salesforce Marketing Cloud] connection exports audience data as files to a storage location, which SFMC then imports.

Steps:

1. In AEP, navigate to Destinations → Catalog → Email Marketing

2. Select Salesforce Marketing Cloud (Files)

3. Connect to the destination by providing your storage location credentials

4. Activate your audiences to the destination

5. Once files land in your storage location, set up a data import job in Salesforce Marketing Cloud to pull the files in

B. API-Based Streaming Destination (B2C Use Cases)

The [!DNL (API) Salesforce Marketing Cloud] destination is ideal for B2C use cases with shorter transactional decision-making cycles. It uses the SFMC Update Contacts API to add contacts and update contact data in real time.

Key characteristics:

  • Streaming/API-based (not file-based)
  • Adds and updates contacts within SFMC segments
  • Best for large datasets representing audience behavior

C. Salesforce Marketing Cloud Account Engagement / Pardot (B2B Use Cases)

The [!DNL Salesforce Marketing Cloud Account Engagement] destination (formerly Pardot) is ideal for B2B use cases — multiple departments, longer sales cycles, and CRM-integrated workflows.

  • Uses the Prospect Upsert by Email API endpoint
  • Adds or updates leads after audience activation
  • Supports lead scoring, nurturing, grading, and campaign segmentation
  • Authenticates via OAuth 2 with Authorization Code

> Important: A V2 version of this destination is currently in Alpha. If you are using the previous version, you must migrate to V2 before January 2026, after which the previous version will be decommissioned.

Option 2: Ingest Salesforce Marketing Cloud Data INTO AEP (Source Connector)

You can bring SFMC data into AEP using the Salesforce Marketing Cloud source connector.

Via the UI

Prerequisites:

  • Working knowledge of XDM (Experience Data Model)
  • Understanding of Real-Time Customer Profile in AEP

Steps:

1. In AEP, navigate to Sources → Catalog → Marketing Automation

2. Select Salesforce Marketing Cloud (V2)

3. Enter your authentication credentials (see the SFMC source overview for prerequisite details)

4. Configure a dataflow to map SFMC fields to XDM schema fields

5. Set your ingestion schedule and activate the dataflow

Via the Flow Service API

You can also connect programmatically using the Flow Service API to create base connections, source connections, target connections, and dataflows.

> Important: The original (V1) Salesforce Marketing Cloud source is deprecated. You must use the V2 source connector. If you have existing dataflows using the V1 connector, migrate to V2 before January 2026.

Choosing the Right Integration

| Use Case | Recommended Integration |

|---|---|

| Send B2C audiences to SFMC contacts in real time | (API) Salesforce Marketing Cloud Destination |

| Send audiences via file batch export | (Files) Salesforce Marketing Cloud Destination |

| B2B lead management with Pardot/Account Engagement | Salesforce Marketing Cloud Account Engagement Destination (V2) |

| Bring SFMC engagement data into AEP for analysis/profile building | Salesforce Marketing Cloud Source (V2) |
