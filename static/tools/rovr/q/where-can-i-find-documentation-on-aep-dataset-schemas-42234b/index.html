What I Know About AEP Schemas and Datasets

  • AEP uses XDM (Experience Data Model) schemas as the foundation for all datasets.
  • For cross-channel and profile use cases, the `_xdm.context.profile__union` schema is commonly used for profile-enabled datasets.
  • Schemas can be referenced when creating datasets via the Catalog Service API using a `schemaRef` object that includes:
  • `id` — the schema URI
  • `contentType` — the schema version descriptor
