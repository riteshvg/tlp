Real-Time Processing

Characteristics:

  • Data is processed and available for reporting within seconds
  • Best for monitoring live site activity and trends

Edge Processing (Partial Information)

Edge Servers process data at the network edge through:

  • Server-to-server APIs (Edge Network Server API)
  • Distributed geographically for faster processing
  • Configured through Datastreams
  • Enable real-time decisioning and personalization
