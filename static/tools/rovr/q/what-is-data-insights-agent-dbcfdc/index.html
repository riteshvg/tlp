What is data insights agent?

Data Insights Agent

The Data Insights Agent is an AI-powered generative conversational agent within Customer Journey Analytics that helps you quickly analyze your data using natural language questions. It builds relevant visualizations and tables automatically based on your actual data and data view components.

What It Does

Data Insights Agent enables you to:

  • Ask natural language questions about your data without manually building visualizations
  • Generate instant visualizations - The agent automatically selects the most appropriate chart type (line graphs, bar charts, donut charts, etc.)
  • Create freeform tables that display your data with relevant dimensions and metrics
  • Reduce time spent on manual analysis and visualization building
  • Make faster, data-driven decisions with quick insights into trends and performance

How It Works

Simply enter a natural language prompt, and the agent will:

1. Interpret your question - Understands what you're asking about your data

2. Map to your data - Translates your prompt into the right dimensions and metrics from your data view

3. Generate results - Creates the most appropriate visualization and freeform table to answer your question

4. Deliver insights - Provides actionable insights you can share with stakeholders

Practical Examples

Example 1: Trend Analysis

  • Prompt: "Trend orders in July"
  • Result: Automatically generates a line graph and freeform table showing orders throughout July

Example 2: Regional Comparison

  • Prompt: "Show revenue by region"
  • Result: Creates a bar chart displaying revenue broken down by customer region

Example 3: Adding Metrics

  • Prompt: "Add profit" (following the previous regional revenue query)
  • Result: Updates the visualization and adds profit as a column in the freeform table

Example 4: Proportional Analysis

  • Prompt: "Proportion of revenue by product category"
  • Result: Generates a donut chart showing the breakdown of revenue by product category
