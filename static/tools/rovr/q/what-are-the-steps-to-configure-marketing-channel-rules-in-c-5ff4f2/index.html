Configuring Marketing Channels in CJA via Derived Fields (Web SDK Path)

Prerequisites:

  • A data view must already be configured in CJA
  • If using the Analytics source connector, Marketing Channel processing rules must already be set up in Adobe Analytics
  • Report suite data must already be imported into AEP via the Analytics source connector (if using that path)

Steps

1. In Customer Journey Analytics, select the data view where you want to add marketing channels.

2. In the data view, select the Components tab.

3. Select Create derived field in the left rail.

4. In the Create derived field dialog box, select Function templates from the drop-down menu.

5. Drag the Marketing channels template onto the blank canvas.

6. Customize the logic for each marketing channel to ensure it matches the logic you use to identify each channel in your Adobe Analytics environment.

  • You can modify the output channel names
  • You can add logic to identify additional channels specific to your organization

7. In the right column, specify a name and a description for the marketing channel derived field.

8. Select Save.
