What is a Edge Server

Edge Server

Based on the Adobe Experience Platform documentation, an Edge Server is part of Adobe's Edge Network infrastructure that processes data at the edge of the network rather than routing everything through a central hub.

Key Characteristics

Location and Purpose:

  • Edge Servers are geographically distributed to process data closer to the source
  • They provide a faster, more efficient alternative to centralized hub-based processing
  • They enable real-time data processing and decisioning at the network edge

How It Works with Adobe Experience Platform

The Edge Network infrastructure allows you to:

  • Send data via the Edge Network Server API - a secure, authenticated server-to-server API
  • Collect and process data server-side before sending it to Experience Platform
  • Configure data flow through Datastreams that determine how data is processed and routed

Related Capabilities

Edge Servers work in conjunction with:

  • Web SDK - for client-side data collection
  • Datastreams - configurations that control data processing at the edge
  • Event Forwarding - for server-side event processing
  • Experience Platform Profile and Query Service - for downstream data activation

This distributed approach allows Adobe Experience Platform to process customer data more efficiently while enabling personalization, audience activation, and real-time insights.
