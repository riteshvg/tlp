What is the Agent Orchestrator?

Adobe Experience Platform Agent Orchestrator

Agent Orchestrator is an intelligent agentic layer built into Adobe Experience Platform that leverages your rich customer data and platform knowledge to power specialized expert agents. It enables complex decision-making and problem-solving tasks at scale with human oversight.

Key Capabilities

Agent Orchestrator works through a conversational AI Assistant interface where you can ask questions and request help using natural language. Here's how it operates:

  • Automatic Agent Activation: When you ask questions, Agent Orchestrator automatically calls upon specialized agents to get you the right answers
  • Conversation Memory: It remembers your conversation history, allowing you to build on previous questions naturally without repeating context
  • Multi-Agent Intelligence: It combines insights from multiple agents to present unified, clear responses
  • End-to-End Workflows: You can complete complex workflows through an intuitive conversational interface without needing to know which agents are working behind the scenes
  • Adaptive Planning: The system understands your goals, creates step-by-step plans, and adjusts its approach based on your feedback

Core Components

AI Assistant Conversational Interface — The primary way you interact with Agent Orchestrator through natural language

Reasoning Engine — Handles decision-making and planning

Specialized Agents — Purpose-built expert agents (like Data Insights Agent) that execute specific tasks

Knowledge Base — Provides access to relevant information from your Experience Platform data

Where You Can Use It

Agent Orchestrator is accessible across multiple Adobe CX Enterprise applications, including:

  • Adobe Journey Optimizer
  • Real-Time CDP
  • Customer Journey Analytics
  • Adobe Experience Platform

Through this single conversational interface, you can analyze trends, identify audiences, validate journey performance, and more—all powered by intelligent agents working behind the scenes.
