What is Adobe Target?

[!DNL Adobe Target] is the [!DNL Adobe Experience Cloud] solution that provides everything you need to tailor and personalize your customers' experience so you can maximize revenue on your web and mobile sites, apps, social media, and other digital channels.

Key Capabilities

Personalization & Optimization

Target helps you create personalized experiences by testing different variations of your content and automatically serving the most effective ones to your visitors. This drives higher conversion rates and revenue.

Machine Learning

[!DNL Target] uses advanced machine learning to intelligently select which experiences work best for each individual visitor based on their profile and behavior patterns similar to other visitors.

Multiple Activity Types

You can create various types of optimization activities including:

  • A/B Tests for comparing content variations
  • Multivariate Tests for testing multiple elements
  • Experience Targeting for specific audiences
  • Automated Personalization (AP)
  • Auto-Allocate to dynamically shift traffic to better performers
  • Auto-Target for advanced personalization

Flexible Implementation

Target supports multiple implementation approaches:

  • Client-side implementations using Adobe Experience Platform Web SDK or at.js
  • Server-side SDKs (Node.js, Java, .NET, Python)
  • Hybrid implementations combining both approaches
  • Mobile apps and email campaigns

Comprehensive Reporting

Built-in reporting helps you understand test results, measure lift, and make data-driven decisions about your optimization efforts.

Target Standard vs. Target Premium

Target Standard provides core optimization features, while Target Premium includes advanced capabilities like Automated Personalization, Auto-Target, and Recommendations—a personalized product recommendation engine.

Getting Started

To begin using [!DNL Adobe Target], you'll need to implement it on your website. The Adobe Target Developer Guide{target=_blank} provides detailed implementation guidance for your specific needs.
