Activating an Audience to a Destination (e.g., Facebook) in Adobe Experience Platform

Prerequisites

Before activating audiences, ensure you have the following:

  • [!UICONTROL View Destinations] and [!UICONTROL Activate Destinations] access control permissions
  • A configured Adobe Experience Platform instance with audiences already built
  • For Facebook specifically:
  • A valid Facebook ad account
  • Audience data must use hashed identifiers (hashed emails, hashed phone numbers, GAID, IDFA, or custom external IDs) — Facebook does not accept PII in clear text

Step 1: Navigate to the Destinations Catalog

1. In the left sidebar, go to Connections → Destinations → Catalog

2. Browse the catalog — for Facebook, look under the Social category

3. Locate the Facebook Custom Audiences destination tile

Step 2: Create a Connection to the Destination

1. Select the Facebook destination tile and click Set up or Activate audiences

2. Authenticate with your Facebook ad account credentials

3. Fill in the destination details:

  • Name: A recognizable name for this destination connection
  • Description: An optional description

4. Select Connect to destination to complete authentication

5. Each Facebook destination in Experience Platform maps 1:1 to a Facebook ad account — you can create multiple destination connections for separate ad accounts

Step 3: Activate Audiences to the Destination

1. Once the destination connection is created, navigate to Connections → Destinations

2. Find your configured Facebook destination and select Activate audiences

3. Select the audiences you want to send to Facebook

4. In the Mapping step, map your identity fields (e.g., hashed email → Facebook's expected identifier)

5. Set your schedule — define start/end dates and export frequency

6. Proceed through the workflow steps and reach the Review screen

Step 4: Review and Finish

1. On the Review screen, verify:

  • Destination connection status
  • Selected audiences and their mapping

2. If everything looks correct, select Finish to activate

3. Each selected audience is sent as a separate call to the destination

Step 5: Verify Activation

  • Navigate to Connections → Destinations → select your destination
  • Use the Destination Monitoring view to track the flow of data and confirm audiences are being exported successfully

Key Notes for Facebook

| Supported Identities | Details |

|---|---|

| Hashed Emails | SHA-256 hashed |

| Hashed Phone Numbers | SHA-256 hashed |

| GAID | Android Advertising ID |

| IDFA | Apple Advertising ID |

| Custom External IDs | Partner-defined IDs |

  • You can create multiple Facebook destinations for different Facebook ad accounts — there is no limit
  • Once activated, you choose which app (Facebook, Instagram, Audience Network, Messenger) to run campaigns against at the placement level inside Facebook Ads Manager
