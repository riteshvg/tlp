What is a datastream in Adobe Experience Platform and how does it route data to different services?

Datastreams in Adobe Experience Platform

What is a Datastream?

A datastream represents the server-side configuration for the Adobe Experience Platform Web and Mobile SDKs. While the SDK's `configure` command handles client-side settings (like the `edgeDomain`), datastreams manage all other configurations on the server.

When you send a request to the Edge Network, the `datastreamId` references the datastream where the data is sent. This allows you to update your server-side configuration without changing your website's code.

How Datastreams Route Data

Core Architecture

1. Data Collection — Your Web SDK or Mobile SDK collects customer experience data

2. Edge Network Reception — Data is sent to the Adobe Experience Platform Edge Network using the datastream ID

3. Service Routing — The datastream configuration determines where and how that data flows to downstream services

4. Enrichment & Transformation — Data can be enriched, validated, and transformed before delivery

5. Destination Delivery — Data routes to Adobe destinations (Analytics, Target, Journey Optimizer, Real-Time CDP) and non-Adobe destinations

The Datastream Configuration Flow

```

Client-side Data Collection (Web/Mobile SDK)

↓

Edge Network (with datastreamId)

↓

Datastream Configuration (Server-side)

↓

Service Routing & Enrichment

↓

Adobe Destinations │ Non-Adobe Destinations

• Analytics │ • Ad platforms

• Target │ • CDPs

• AJO │ • Analytics tools

• RTCDP │ • Marketing platforms

```

Key Datastream Capabilities

1. Multi-Destination Routing

A single datastream can route the same data to multiple services simultaneously:

  • Adobe Experience Platform — Feeds the Real-Time Customer Profile and Data Lake
  • Adobe Analytics — For analytics reporting
  • Adobe Target — For personalization and A/B testing
  • Adobe Journey Optimizer — For omnichannel messaging
  • Non-Adobe destinations — Third-party platforms via event forwarding and configured integrations

2. Data Prep for Data Collection

Transform and validate data before it reaches downstream services:

  • Map incoming data to your XDM schema
  • Add calculated fields
  • Delete unwanted fields
  • Validate data quality

3. Bot Detection

Configure automated bot detection rules to filter out non-human traffic before data is processed.

4. Event Forwarding

Send events directly to server-side destinations (like AWS, Google Ads, webhooks) without storing data in Experience Platform first.

5. Datastream Overrides

Override datastream settings at the request level for:

  • Different property IDs
  • Report suite configurations
  • Destination settings
  • Analytics reporting data

6. Enhanced Encryption (for sensitive data)

  • Encrypt sensitive customer data (PHI, PCI, etc.) before transmission
  • Decrypt only at authorized destinations

Managing Datastreams

You can create and manage datastreams by selecting [!UICONTROL Datastreams] in the left navigation within:

  • The Adobe Experience Platform UI, or
  • The Data Collection UI

For detailed configuration instructions, see the datastream configuration guide.

Important: Sensitive Data Handling

> ⚠️ Important: If your organization collects sensitive data (Protected Health Information, Payment Card Information, etc.), datastreams provide three methods to handle it securely:

>

> 1. Enhanced encryption — Encrypt data in transit

> 2. Data governance — Policy-based controls on data use

> 3. Audit logs — Track all data access and modifications

Consult your company's legal and compliance teams when handling sensitive customer data, as regulatory requirements (like HIPAA, GDPR, CCPA) may apply.

Summary

Datastreams are the central hub for server-side data routing in Adobe Experience Platform. They decouple your client-side code from backend configurations, enabling you to:

✅ Route data to multiple Adobe and non-Adobe services

✅ Transform and validate data in real time

✅ Update configurations without code changes

✅ Control sensitive data handling securely

✅ Activate customer data to personalization and marketing destinations
