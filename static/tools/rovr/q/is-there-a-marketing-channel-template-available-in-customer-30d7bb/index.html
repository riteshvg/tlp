Adobe-Provided Marketing Channel Templates in CJA

Pre-Built Adobe Templates in Analysis Workspace

The documentation confirms that Adobe provides pre-built templates in Analysis Workspace for marketing channel analysis. The following are explicitly mentioned:

| Template Name | Location | Purpose |

| --- | --- | --- |

| Channel Overview Report | Marketing channels > Channel overview report | Shows how visitors arrive on your site using custom attribution — helps understand which marketing channels are most effective |

| First Touch Channel | Marketing channels > First touch channel | Views the first marketing channel a visitor matches during their engagement period (30 days by default) |

| First Touch Channel Detail | Marketing channels > First touch channel detail | Provides detail about the first marketing channel match (e.g., which search engine or keyword for paid search) |

| Last Touch Channel | Marketing channels > Last touch channel | Views the most recent channel matched before conversion |

Derived Fields Marketing Channel Function Template

Separately, Adobe also provides a Marketing Channels function template specifically for creating derived fields within Data Views:

1. In your Data View, select the [!UICONTROL Components] tab

2. Select [!UICONTROL Create derived field] in the left rail

3. Select [!UICONTROL Function templates] from the drop-down menu

4. Drag the [!UICONTROL Marketing channels] template onto the blank canvas

This template provides pre-built logic for defining marketing channel rules that can be customized to match your organization's needs.
