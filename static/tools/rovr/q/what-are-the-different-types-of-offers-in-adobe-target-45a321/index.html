What are the different types of offers in Adobe Target?

Types of Offers in Adobe Target

Adobe Target supports two main types of offers that you can manage and use in your activities:

1. Code Offers

Code offers allow you to deliver custom HTML, JavaScript, or other code-based content. These are useful for:

  • Creating personalized HTML experiences
  • Implementing custom logic and dynamic content
  • Integrating with third-party systems through code

You can manage code offers through the Offers > Code Offers section in Target.

2. Image Offers

Image offers let you use image assets as part of your personalized experiences. These are useful for:

  • Displaying product images
  • Visual promotions and banners
  • Picture-based recommendations

You can upload and manage image offers through the Offers > Image Offers section in Target.

Dynamic Content in Offers

Both types of offers support dynamic data functionality, which allows you to:

  • Pass visitor profile information into offers
  • Display activity-related data (activity name, experience name)
  • Create personalized content based on visitor interests, past behavior, and profile data

Example use case:

Instead of creating separate offers for every product in your catalog, you can create a single dynamic offer that reads the "last product purchased" from a visitor's profile and displays a personalized recommendation message.

Both code and image offers can be organized, searched, filtered, copied, edited, and deleted within the Target Offers library to help you efficiently manage your content for personalization campaigns.
