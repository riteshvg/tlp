What Causes Missing Email Delivery in AJO

Based on the documentation available, email delivery failures in Adobe Journey Optimizer can occur for multiple reasons. Here are the key causes:

Email Delivery Errors

The Adobe Journey Optimizer system tracks all email delivery errors with specific error labels, types, and technical values. These errors are recorded in the AJO Message Feedback Event Dataset, which contains:

  • Message delivery logs from all Journey Optimizer sends
  • Feedback records from email ISPs on bounces
  • Complete delivery failure information

Common Delivery Failure Categories

Email deliveries can fail due to:

1. ISP Feedback Issues – Email service providers rejecting or bouncing messages

2. Bounce-related errors – Hard bounces (invalid addresses) or soft bounces (temporary issues)

3. Configuration problems – Email channel not properly set up

4. Content validation failures – Email content not meeting delivery requirements

5. Suppression and exclusion rules – Profiles on suppression lists or excluded from sends

Accessing Delivery Error Details

To diagnose missing email delivery:

  • Check the Message Feedback Event Dataset – This contains complete delivery logs and ISP feedback records for all messages sent from Journey Optimizer
  • Review journey execution reports – See which messages were sent, failed, or were blocked
  • Examine error codes and descriptions – Each delivery failure includes an error label, type, and technical value for troubleshooting

Prerequisites for Email Sending

Before emails can be delivered successfully:

  • Email channel must be configured – Ensure basic email settings are established in Journey Optimizer
  • Sender authentication – SPF, DKIM, and DMARC records should be verified
  • Subdomain delegation – Verify proper subdomain configuration for sending

For detailed information about specific error codes and their meanings, consult the complete email delivery errors reference in the Message Feedback Event Dataset documentation.
