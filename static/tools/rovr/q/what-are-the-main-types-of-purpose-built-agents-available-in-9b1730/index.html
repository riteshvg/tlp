Data Insights Agent

The Data Insights Agent is designed to help you analyze and visualize data through natural language queries. It can respond to common prompts such as:

  • Analyzing trends (e.g., "Show me profits in [Month]")
  • Comparing metrics across dimensions (e.g., "Show revenue by region")
  • Identifying top performers (e.g., "What were my top 10 SKUs by profit?")
  • Calculating proportions and totals

The agent returns appropriate visualizations based on your query, including:

  • Line charts for trends over time
  • Bar charts for comparisons across categories
  • Donut charts for proportions
  • Summary numbers for totals
