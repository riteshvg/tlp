How do I send personalized email offers using AJO decisioning?

Sending Personalized Email Offers Using AJO Decisioning

Adobe Journey Optimizer Decisioning allows you to automatically select and deliver the most relevant offer to each email recipient based on their profile attributes, eligibility rules, and constraints. Here is a complete end-to-end guide.

Prerequisites

  • Access to Adobe Experience Platform and Journey Optimizer
  • Basic understanding of AEP concepts (Profiles, Audiences, Datasets)
  • Email channel configured in Journey Optimizer
  • Familiarity with Decision Management concepts (offers, decisions, selection strategies)

High-Level Steps

1. Ingest profiles / create audiences

2. Create decisioning components (offers, selection strategies, ranking formulas)

3. Build and configure an email campaign

4. Insert a decision into the email

5. Publish and execute

Step 1: Ingest Profiles and Create Audiences

Ensure your customer profiles exist in AEP with the relevant attributes you want to use for offer targeting (e.g., annual income, loyalty status). Create the appropriate audience segments that will be targeted by your campaign.

Step 2: Create Personalized Offers

Prerequisite: You must have the Decision Management feature available in your Journey Optimizer instance.

1. In the Journey Optimizer UI, select Offers from Decision Management in the left rail.

2. In the Offers screen, select Browse to see the list of offers.

3. Select Create offer.

4. In the New offer dialog, select Personalized offer and click Next.

5. In the Details step:

  • Enter a Name for the offer (e.g., `Luma - Juno Jacket`).
  • Enter a Start date and time and an End date and time. Only offers within these dates are selected by the Decisioning engine.
  • Select Next.

6. Add representations (the actual content — HTML, image, text) for each placement.

7. Define eligibility rules and constraints (e.g., frequency capping, profile attributes like loyalty tier or income bracket).

8. Save the offer.

Repeat this process to create multiple personalized offers for different audience segments.

Step 3: Configure a Selection Strategy and Decision

  • Create a Selection Strategy that groups your offers and defines a Ranking Formula (how the engine scores and ranks offers per profile).
  • Create a Decision that combines placements with your selection strategy, so the engine knows which offers to evaluate for which content slot.

Step 4: Create an Email Campaign and Insert the Decision

1. In Journey Optimizer, create a new Campaign and select Email as the action channel.

2. Define basic campaign settings (name, audience, schedule).

3. Open the Email Designer to design your message.

4. Place your cursor in the area of the email where you want the personalized offer to appear.

5. Use the personalization/offers component to insert a decision:

  • Select the decision you configured in Step 3.
  • The Decisioning engine will evaluate each recipient's profile at send time and insert the winning offer's content.

6. You can also add a fallback offer to ensure every recipient sees something even if no personalized offer qualifies.

> Important: If changes are made to an offer decision being used in a journey's message, you must unpublish the journey and republish it so the changes are incorporated.

Step 5: Preview, Test, and Publish

  • Use the Preview feature in the Email Designer to simulate how different profiles will see different offers.
  • Once satisfied, activate the campaign.
  • At send time, Journey Optimizer's Decisioning engine automatically selects the best offer for each recipient based on their profile and the configured rules.

Key Concepts Summary

| Component | Purpose |

|---|---|

| Personalized Offer | Customizable marketing message with eligibility rules and constraints |

| Selection Strategy | Defines which offers compete and how they are ranked |

| Ranking Formula | Scores offers per profile to determine the best match |

| Decision | Ties placements + selection strategies together |

| Fallback Offer | Ensures delivery when no personalized offer qualifies |

This end-to-end flow ensures that each customer receives the most relevant offer in their email, driven automatically by their real-time profile data.
