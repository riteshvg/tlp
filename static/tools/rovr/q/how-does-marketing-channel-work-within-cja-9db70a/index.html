Marketing Channels in Customer Journey Analytics

How Marketing Channels Work in CJA

Marketing channels in Customer Journey Analytics function differently depending on how your data flows into CJA. There are two primary paths:

1. Via the Analytics Source Connector

When using the Analytics source connector, marketing channel data flows into CJA directly through that connector. However, there are important caveats:

  • Marketing Channel rules are configured in traditional Adobe Analytics
  • Some rules are not supported in CJA when coming through the source connector
  • Certain settings behave differently, including:

Override Last-Touch Channel

This is a key behavioral difference to be aware of:

  • The Override Last-Touch Channel setting (configured in the Marketing Channel Manager in Adobe Analytics) is ignored by CJA
  • This means broad channels like Direct or Internal may incorrectly receive attribution credit in undesired ways

Recommended mitigations:

  • Remove channels where Override Last-Touch Channel is unchecked
  • Delete the Direct marketing channel in the Marketing Channel Manager and rely on CJA's "No value" dimension item instead (which can be renamed to "Direct" in a data view)
  • Create a marketing channel classification, mapping each value to itself except for channels you want to exclude, then use that classification dimension in your data view instead of `channel.typeAtSource`

2. Via the Experience Platform Web SDK

When using the Web SDK, marketing channel data does not flow in automatically. Instead, you recreate marketing channels using Derived Fields within a CJA Data View:

Steps to Create Marketing Channels as Derived Fields

Prerequisites: You must have a data view already configured in CJA.

1. In Customer Journey Analytics, select the data view where you want to add marketing channels.

2. In the data view, select the Components tab.

3. Select Create derived field in the left rail.

4. In the Create derived field dialog box, select Function templates from the drop-down menu.

5. Drag the Marketing channels template onto the blank canvas.

6. Customize the logic for each marketing channel to match the rules you use in your Adobe Analytics environment.

7. Optionally, modify output channel names or add logic for additional channels specific to your organization.

8. In the right column, specify a name and description for the marketing channel derived field.

9. Select Save.

> Note: Using a default marketing channel setup takes only a few minutes, while a highly customized setup may take several hours.

Key Differences vs. Traditional Adobe Analytics

| Behavior | Adobe Analytics | Customer Journey Analytics |

|---|---|---|

| Channel rules location | Marketing Channel Manager | Derived Fields in Data View |

| Override Last-Touch Channel | Respected | Ignored |

| Attribution model | Session-based | Configurable per use case |

| "No value" handling | Limited | Can be renamed/excluded in data view |
