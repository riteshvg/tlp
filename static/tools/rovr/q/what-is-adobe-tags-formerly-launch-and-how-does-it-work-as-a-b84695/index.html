What is Adobe Tags (formerly Launch) and how does it work as a tag management system?

Adobe Tags Overview

Adobe Tags (formerly known as Adobe Experience Platform Launch) is a modern tag management system built into Adobe Experience Platform. It's the next generation of tag management capabilities from Adobe, designed to simplify how you deploy and manage analytics, marketing, and advertising solutions.

What Tags Does

Tags enable you to:

  • Deploy client-side web products using pre-built integrations called extensions
  • Dynamically deliver configuration to update implementations in native mobile applications
  • Consistently capture, define, manage, and share data across marketing and advertising products—both Adobe solutions and third-party vendors

How It Works as a Tag Management System

Tags function as an advanced code and configuration delivery system with these core capabilities:

Centralized Management

Tags allows you to centrally manage all JavaScript related to analytics, marketing, and advertising solutions used on your desktop and mobile sites. For example, if you deploy Adobe Analytics, Tags manages the AppMeasurement JavaScript library, populates variables, and fires requests—all from one place.

Modular & Efficient

The system is fully modular and minified:

  • Only the code you need is included in your library
  • Everything is minified for optimal performance
  • The result is a fast, compact implementation

Condition Evaluation & Action Execution

Tags evaluates conditions and executes actions to efficiently deploy client-side libraries and products. This rule-based approach gives you powerful control over when and how your tags fire.

Extension-Based Architecture

Think of Tags as an operating system, and extensions are the apps you install to achieve your tasks. Extensions are pre-packaged bundles of code (JavaScript, HTML, and CSS) that extend Tags functionality. You can:

  • Browse and install extensions from the extension marketplace
  • Configure extensions to suit your needs
  • Build custom extensions for proprietary solutions
  • Manage integrations with independent software vendors

Key Benefits

  • Faster time to value – Deploy solutions quickly without custom coding
  • Trustworthy data – Centralized collection, organization, and delivery through data elements
  • Compelling experiences – Integrate data and marketing technology using the rule builder

Pricing

Tags is available at no additional charge to all Adobe Experience Cloud customers as an included value-add feature.
