August 2025 Release

Release date: August 19, 2025

Alerts — Streaming Throughput Capacity Alerts

New Feature: Three new alerts allow you to subscribe and configure alerts to proactively manage streaming throughput capacity:

  • Alert when streaming throughput reaches 80% capacity
  • Alert when streaming throughput reaches 90% capacity
  • Alert when streaming throughput exceeds capacity limits

These help you monitor and optimize your streaming data ingestion performance.
