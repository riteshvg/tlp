79. How do you insert dynamic subject line content?

How to Insert Dynamic Subject Line Content

In Adobe Journey Optimizer, you can create dynamic subject lines using conditional rules and the personalization editor. Here's how:

Using the Personalization Editor

1. Open your email message in the Email Designer

2. Click on the Subject line field to edit it

3. Click the personalization icon (usually a `{}` symbol) to open the personalization editor

4. Build your dynamic content using:

  • Profile attributes – Reference customer data like first name, location, preferences
  • Contextual events – Use real-time data from journey events
  • Conditional rules – Apply `if/then` logic to display different subject lines based on criteria

Example Dynamic Subject Lines

  • `Hello {{profile.person.name.firstName}}, your order is ready!` – Personalizes with first name
  • `{{if profile.person.address.countryCode == 'US'}}Free shipping available!{{else}}International shipping applied{{/if}}` – Shows different content based on country

Using Conditional Rules

1. Create a conditional rule in the personalization editor using the rule builder

2. Define conditions based on:

  • Profile attributes (age, location, preferences, etc.)
  • Audience membership
  • Event data

3. Set different subject line variants for each condition branch

4. Add a fallback subject line for when conditions aren't met

Key Benefits

  • Increased engagement – Personalized subject lines improve open rates
  • Relevance – Content adapts to each recipient's context
  • Consistency – Save conditional rules for reuse across campaigns and journeys

The personalization editor provides a visual interface to build these expressions without requiring coding knowledge.
