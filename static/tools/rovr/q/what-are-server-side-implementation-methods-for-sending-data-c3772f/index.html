What are server-side implementation methods for sending data to Adobe Analytics?

I don't have information on server-side implementation methods in the Adobe Experience League documentation provided.

The documentation I have access to focuses primarily on client-side implementation methods for Adobe Analytics, which include:

  • AppMeasurement (JavaScript)
  • Adobe Experience Platform Web SDK
  • Tags in Adobe Experience Platform Data Collection
  • APIs (Data Insertion API, Bulk Data Insertion API, Data Collection API)

However, I do have information about Server-Side Forwarding, which is a feature that allows data collected in Adobe Analytics to be forwarded to other Adobe CX Enterprise solutions (like Adobe Audience Manager) in real-time. This is different from a primary server-side data collection method.

If you're looking for information about:

  • Primary server-side data collection methods (sending data directly from your server to Adobe Analytics)
  • Backend implementations for Analytics
  • Server-to-server data transmission approaches
