## ğŸ“ Writing Reflections

### Using the CMS

1. Visit `/admin`
2. Click "New Reflection Post"
3. Fill in the form:
   - Title
   - Date
   - Tags & Category
   - Featured Image (optional)
   - Excerpt
   - Body (write your post)
4. Click "Save"

### Post Structure

Each reflection includes:

```yaml
title: "Your Reflection Title"
date: 2025-02-20 00:00:00
author: "Ritesh Gupta"
tags: ["tag1", "tag2"]
category: "Personal"
thumbnailImage: "/images/reflections/image.jpg"
excerpt: "Brief summary"
draft: false
```

### Categories
