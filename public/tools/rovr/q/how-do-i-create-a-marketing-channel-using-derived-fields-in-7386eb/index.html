How do I create a marketing channel using Derived Fields in CJA?

Creating a Marketing Channel Using Derived Fields in CJA

Prerequisites

  • You must be a CJA Administrator
  • You need an existing Data View to add the derived field to
  • Your underlying datasets should include fields you want to use as channel signals (e.g., referrer URL, UTM parameters, page URL, tracking codes)

Step-by-Step Instructions

1. In Customer Journey Analytics, select the Data View where you want to add marketing channels.

2. In the data view, select the [!UICONTROL Components] tab.

3. Select [!UICONTROL Create derived field] in the left rail.

4. In the [!UICONTROL Create derived field] dialog box, select [!UICONTROL Function templates] from the drop-down menu.

5. Drag the [!UICONTROL Marketing channels] template onto the blank canvas.

6. Customize the logic for each marketing channel to ensure it matches the logic you use to identify each channel in your Adobe Analytics environment (if migrating) or define your new channel rules.

7. Optionally, modify the output channel names or add logic to identify additional channels specific to your organization.

8. In the right column, specify a name and a description for the marketing channel derived field.

9. Select [!UICONTROL Save].

Tips for Customizing Channel Logic

  • Replicate existing Adobe Analytics rules: If you're migrating from Adobe Analytics, match the same conditions you had in your Marketing Channel processing rules
  • Add organization-specific channels: You can extend the default template with additional conditions unique to your business
  • Use any available XDM field: Channel conditions can be based on referrer URL, UTM parameters, page URL, query string parameters, or any other field in your dataset
  • Order matters: Rules are evaluated top-to-bottom, so place more specific conditions before broader ones

After Saving

Once the derived field is saved, it becomes available as a dimension component in the data view. You can then:

  • Use it in Analysis Workspace freeform tables
  • Apply different attribution models (first touch, last touch, custom)
  • Combine it with other dimensions and metrics for channel performance reporting
