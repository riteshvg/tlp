What is an Audience Agent? How does it work?

Audience Agent

The Audience Agent is a specialized expert agent within Adobe Experience Platform that streamlines audience management through conversational interactions. It enables you to create, optimize, analyze, and manage audiences using natural language—without needing to navigate complex UI workflows.

Key Capabilities

The Audience Agent lets you:

  • Create audiences conversationally — Describe the audience you need, and the agent builds it for you
  • View audience insights — Understand audience composition, size, and characteristics
  • Detect audience issues — Identify significant size changes and duplicate audiences
  • Explore your audience inventory — Discover and review existing audiences
  • Optimize audiences — Get recommendations for improvements
  • Retrieve audience metrics — Access audience size and performance data

How It Works

The Audience Agent operates through a step-by-step conversational process:

Example Workflow: Creating an Audience

1. You make a request via natural language in AI Assistant. For example:

```

Create an audience that combines people with an average download usage per month

of over 2000 GB and a preferred genre of sci-fi or fantasy.

```

2. Agent generates a plan — It creates a step-by-step plan for building the audience based on your criteria

3. You review and confirm — You review the plan, audience definition, and segment size estimation at each step

4. Audience is created — The agent builds the audience in Experience Platform

5. You access the result — The audience becomes available for use (typically within 24 hours for full availability to AI Assistant)

Supported Use Cases

The Audience Agent handles:

  • Audience creation — Build new segments based on data attributes and behaviors
  • Audience health monitoring — Proactively detect issues and suggest solutions
  • Duplicate detection — Identify redundant audiences to clean up your inventory
  • Audience insights — Analyze audience trends and characteristics

Permissions Required

To use the Audience Agent, you need:

  • View Segments — Allows you to use the Audience Agent to view audience insights directly in AI Assistant
  • Manage Segments — Allows you to create new audiences directly in AI Assistant

Integration

The Audience Agent integrates securely across B2B and B2C applications, making audience management a seamless part of your customer experience workflows.
