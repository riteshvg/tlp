What does AJO Push Tracking Experience Event dataset contain

AJO Push Tracking Experience Event Dataset

Based on the available documentation, the AJO Push Tracking Experience Event dataset is designed to capture and store tracking data related to push notification interactions within Adobe Journey Optimizer.

While the retrieved context doesn't provide exhaustive details about every field in this specific dataset, here's what we know about its purpose:

Primary Function

This dataset captures push notification experience events, which includes tracking data for push messages sent through Journey Optimizer journeys and campaigns.

Typical Data Captured

Push tracking datasets generally store:

  • Delivery events - When push notifications are sent to devices
  • Interaction events - User engagement such as opens, clicks, and dismissals
  • Error/failure events - Bounce backs or failed delivery attempts
  • Timestamp data - When each event occurred
  • Profile identifiers - To connect events to customer profiles
  • Message identifiers - To link events back to specific push campaigns or journey steps

How to Access This Data

Journey Optimizer automatically stores push tracking events, and you can access them through:

1. Query Service - Use SQL queries to analyze the dataset

2. Customer Journey Analytics (CJA) - For advanced analytics and cross-channel analysis

3. Journey Optimizer's built-in reporting - For real-time push performance metrics

4. APIs - For programmatic access to event data

This data integrates with your broader customer profile and journey analytics to help you understand push notification performance and customer engagement patterns.
