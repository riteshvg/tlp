65. What channels are supported by AJO?

Communication Channels Supported by Adobe Journey Optimizer

Adobe Journey Optimizer provides a comprehensive set of outbound and inbound channels to help you connect with customers across their preferred touchpoints. Here's the complete breakdown:

All Supported Channels

Outbound Channels (Proactive Messaging)

These channels allow you to initiate communication with customers:

1. Email — Send personalized email messages

2. SMS — Text message communications

3. Push Notifications — Mobile app notifications (iOS & Android)

4. iOS Live Activities — Time-sensitive, interactive notifications on iOS 16.1+

5. Direct Mail — Generate extraction files for physical mail campaigns

6. WhatsApp — Message customers via WhatsApp Business API

7. Line — Reach customers on the Line messaging platform

Inbound Channels (Contextual, Customer-Initiated)

These channels deliver experiences within customer-initiated interactions:

8. In-App Messaging — Contextual messages within mobile or web apps

9. Web — Personalized web experiences and landing pages

10. Code-Based Experiences — Server-side personalization via SDKs

11. Content Cards — Rich, dismissible content delivered in apps

Additional Capabilities

12. Custom Actions / Webhooks — Call third-party APIs and external systems

13. RCS Messaging — Rich Communication Services (latest innovation)

Channel Descriptions

1. Email ✅ Fully Supported

Best for: Newsletter campaigns, transactional confirmations, promotional offers, lifecycle messaging

  • Rich HTML support with dynamic content
  • Personalization using profile and event attributes
  • A/B testing and experimentation
  • Delivery optimization and best time send
  • Frequency capping and compliance management

2. SMS ✅ Fully Supported

Best for: Time-sensitive alerts, OTP codes, appointment reminders, urgent notifications

  • 160-character SMS or longer MMS
  • Global reach with carrier partnerships
  • Compliance with SMS regulations (GDPR, TCPA)
  • Delivery and engagement tracking

3. Push Notifications ✅ Fully Supported

Best for: App engagement, time-sensitive alerts, user retention

  • iOS and Android support
  • Rich media (images, videos, GIFs)
  • Deep linking to app content
  • Behavior-triggered delivery
  • In-app action buttons

4. iOS Live Activities ✅ Limited Support

Best for: Real-time, time-sensitive updates (sports scores, ride status, order tracking)

  • API-triggered campaigns only
  • Requires iOS 16.1+
  • Live, evolving notifications on lock screen
  • Real-time data updates without app launch

5. In-App Messaging ✅ Fully Supported

Best for: Contextual guidance, feature announcements, onboarding, in-app promotions

  • Web and mobile app support
  • Multiple layout templates (banner, modal, fullscreen, carousel)
  • Event-triggered or scheduled delivery
  • A/B testing capabilities
  • Dismissible by user

6. Direct Mail ✅ Fully Supported

Best for: High-value campaigns, personalized catalogs, legal documents, premium offers

  • Generates extraction files for third-party mail providers
  • Batch or event-triggered delivery
  • Postal address personalization
  • Includes decisioning for offer variants
  • File routing to external providers

7. Web ✅ Fully Supported

Best for: Personalized landing pages, web experiences, site-wide messaging

  • Personalization at the page level
  • A/B testing and experimentation
  • Dynamic content blocks
  • Server-side rendering via Web SDK

8. Code-Based Experiences ✅ Fully Supported

Best for: Developer-driven personalization, mobile app experiences, server-side personalization

  • Delivered via Mobile SDK (iOS/Android) or Web SDK
  • Full control over rendering
  • Custom logic and conditional display
  • Rich personalization data

9. Content Cards ✅ Fully Supported

Best for: In-app feed content, scrollable recommendation cards, rich media galleries

  • Appear in a scrollable feed within apps
  • Dismissible by users
  • Images, titles, descriptions, CTAs
  • Event or audience-triggered
  • User engagement tracking

10. WhatsApp ✅ Fully Supported

Best for: Customer service, appointment confirmations, order updates, personalized offers

  • WhatsApp Business API integration
  • Template-based messages (pre-approved by WhatsApp)
  • Customer-initiated conversations
  • Two-way messaging capabilities

11. Line ✅ Fully Supported

Best for: Engagement in Asia-Pacific markets, rich messaging, loyalty programs

  • Line Official Account integration
  • Template and rich menu support
  • Popular in Japan, Taiwan, South Korea, Thailand

12. Custom Actions / Webhooks ✅ Fully Supported

Best for: Third-party integrations, external API calls, custom business logic

  • REST API calls to any external system
  • Custom authentication (OAuth, API key, basic auth)
  • Request and response payload mapping
  • Feedback webhooks for inbound data
  • Available in both journeys and campaigns

Omnichannel Strategy with AJO

Single Journey, Multiple Channels

  • Create a single customer journey that spans multiple channels
  • Use conditions and audience data to route customers to the right channel
  • Respect customer channel preferences stored in their profile

Example Workflow

```

Customer enters journey

↓

Branch 1: Email preferred → Send email

↓

Branch 2: SMS preferred → Send SMS

↓

Branch 3: Push preferred → Send push notification

↓

All paths converge → Next journey step

```

Cross-Channel Consistency

  • Maintain message consistency across channels while respecting channel-specific formats
  • Use dynamic content blocks to adapt messages to channel capabilities
  • Track engagement and conversions across all channels in unified reporting
