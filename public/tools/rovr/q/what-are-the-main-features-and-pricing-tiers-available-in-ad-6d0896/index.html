What are the main features and pricing tiers available in Adobe Target?

Adobe Target Features and Pricing Tiers

Pricing Tiers

Adobe Target is available in two main editions:

Target Standard

  • Provides core A/B testing and personalization capabilities
  • Best for organizations starting with optimization
  • Includes basic activity types and reporting

Target Premium

  • Includes all Standard features plus advanced capabilities
  • Recommended for enterprises needing sophisticated personalization
  • Provides access to premium features like Automated Personalization and advanced targeting

Main Features by Tier

Available in Both Standard & Premium

Core Testing & Optimization

  • A/B Testing — compare two or more variations
  • Multivariate Testing — test multiple elements simultaneously
  • Experience Targeting — deliver different experiences to specific audiences
  • Basic audience segmentation and targeting

Implementation Options

  • Web SDK integration
  • at.js JavaScript library
  • Server-side SDKs (Node.js, Java, .NET, Python)
  • Mobile app support

Reporting & Analytics

  • Built-in reporting dashboard
  • Conversion tracking and lift measurement
  • Traffic allocation insights
  • Visitor behavior data

Premium-Only Features

Advanced Personalization

  • Automated Personalization (AP) — machine learning automatically tests hundreds of experience combinations and serves the best ones to each visitor
  • Auto-Target — uses advanced machine learning to identify the best experience for each visitor based on their profile and similar visitors' behavior
  • Auto-Allocate — automatically shifts traffic to better-performing variations in real-time

Recommendations

  • Personalized product recommendation engine
  • Recommendation algorithms (collaborative filtering, content-based, hybrid)
  • Dynamic content personalization

Advanced Audience Features

  • More sophisticated audience creation and segmentation
  • Behavioral audience targeting
  • Enhanced audience analytics

Additional Capabilities (Recent Enhancements)

Choosing Your Tier

Choose Standard if you need:

  • Basic A/B and multivariate testing
  • Simple audience targeting
  • Cost-effective optimization solution

Choose Premium if you need:

  • Advanced machine learning and automation
  • Product recommendations
  • Complex personalization scenarios
  • Enterprise-scale optimization

For detailed pricing information specific to your organization, contact your Adobe account representative, as pricing varies based on traffic volumes and specific requirements.
