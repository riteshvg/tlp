What is the use of productListItems object in an ecommerce AEP Web SDK implementation?

The productListItems Object in Adobe Experience Platform Web SDK

Purpose

The `productListItems` object is used in Adobe Experience Platform Web SDK to track product information during ecommerce events. It contains an array of products associated with commerce interactions, allowing you to send detailed product-level data alongside commerce events.

Common Use Cases

The `productListItems` object is used for tracking:

1. Product Views — When customers view product details

2. Product List Adds — When items are added to cart

3. Checkouts — When items proceed through the checkout process

4. Purchases — When transactions are completed

5. Other commerce events — Returns, removals, or any product-related commerce action

Object Structure

Each product in the `productListItems` array can include fields such as:

  • `SKU` — Stock Keeping Unit (product identifier)
  • `name` — Product name
  • `quantity` — Number of units
  • `priceTotal` — Total price for that product line item
  • `productAddMethod` — How the product was added (e.g., "Add to Cart Button", "Add-on")

Practical Examples

Product Views Example

```javascript

alloy("sendEvent",{

"xdm":{

"commerce":{

"productViews":{

"value":1

}

},

"productListItems":[

{

"SKU":"HT105",

"name":"The Big Floppy Hat"

},

{

"SKU":"HT104",

"name":"The Small Floppy Hat"

}

]

}

});

```

Product List Adds Example

```javascript

alloy("sendEvent",{

"xdm":{

"commerce":{

"productListAdds":{

"value":1

}

},

"productListItems":[

{

"SKU":"HT105",

"name":"The Big Floppy Hat",

"quantity":1,

"priceTotal":29.99,

"productAddMethod":"Add to Cart Button"

},

{

"SKU":"HT104",

"name":"The Small Floppy Hat",

"quantity":1,

"priceTotal":9.99,

"productAddMethod":"Add-on"

}

]

}

});

```

Key Benefits

  • Multi-product tracking — Send data for multiple products in a single event
  • Granular analysis — Analyze product-level performance, pricing, and quantities
  • Commerce funnel visibility — Track products through the entire purchase journey
  • Cross-solution compatibility — Data flows to Experience Platform and connected solutions for analysis and personalization

The `productListItems` object is essential for comprehensive ecommerce analytics and enables you to understand product performance across all stages of the customer purchase journey.
