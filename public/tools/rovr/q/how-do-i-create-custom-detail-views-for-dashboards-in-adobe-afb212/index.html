How do I create custom detail views for dashboards in Adobe Analytics?

Creating Custom Detail Views for Dashboards in Adobe Analytics

Custom detail views allow you to refine the story you tell your stakeholders through mobile scorecard projects in Adobe Analytics dashboards.

▶ Watch: Create Custom Detail Views for Dashboards

Overview

Custom detail views are a feature of Adobe Analytics Dashboards (mobile scorecards). They let you customize what executives and stakeholders see when they tap into a tile on a mobile scorecard, including the ability to add custom text to provide narrative context alongside your data.

Key Customization Options

When building detail views for a mobile scorecard tile, you can customize:

  • Visualizations — Choose which chart types and breakdowns appear in the detail view for each scorecard tile
  • Text blocks — Add descriptive or explanatory text to guide stakeholders through the insight being presented
  • Segments and filters — Refine the data shown in the detail view to focus the story
  • Date ranges — Control the time periods displayed within the detail view

How to Access Detail View Settings

1. Open Adobe Analytics and navigate to the Workspace area

2. Open or create a Mobile Scorecard project

3. Click on a scorecard tile to select it

4. In the tile editor, locate the Detail View configuration options

5. Add or modify visualizations, text, and breakdowns as needed to shape the narrative

6. Save the scorecard project to preserve your changes

Best Practices

  • Use custom text to explain what the data means in plain language for non-analyst stakeholders
  • Keep detail views focused — include only the visualizations that directly support the tile's key metric
  • Use breakdowns in detail views to let stakeholders explore dimensions without overwhelming the top-level scorecard

This feature is specifically designed to help you craft a compelling, guided data story for mobile dashboard consumers such as executives and business owners.
