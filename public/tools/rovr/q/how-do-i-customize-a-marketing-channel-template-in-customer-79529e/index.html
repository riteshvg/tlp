Customizing a Template in Customer Journey Analytics

Prerequisites

  • Access to Customer Journey Analytics
  • Access to Analysis Workspace
  • Appropriate permissions to save projects

Step-by-Step: Customize a Template and Save as a Project

1. In Customer Journey Analytics, select the [!UICONTROL Workspace] tab

2. Select the [!UICONTROL Templates] tab

3. Select the template you want to customize (e.g., a marketing channel template such as Channel Overview Report)

4. Once the template opens in Analysis Workspace, do any of the following to customize it:

  • Drag one or more segments into the Segment drop zone at the top
  • Change the date range by going to the calendar at the top-right
  • Add dimension breakdowns to explore channel data in more detail
  • Drag in additional metrics relevant to your marketing channel analysis
  • Modify existing visualizations or add new ones (bar charts, line graphs, etc.)
  • Add additional panels such as "Channel Performance Overview" or "Time Parting Analysis"

5. When satisfied with your customizations, select [!UICONTROL Project] > [!UICONTROL Save]

> Important: Saving changes to a template saves it as a new project — it does not modify the original template. The original template remains unchanged and available for future use.

Creating a Custom Marketing Channel Template for Organizational Use

If you are an admin and want to create a reusable marketing channel template for your organization:

▶ Watch: Create custom templates in Customer Journey Analytics

The documentation describes this process as:

1. Set up a project with relevant panels (e.g., "Channel Performance Overview", "Time Parting Analysis")

2. Configure the panels with appropriate marketing channel dimensions and metrics

3. Save it as a template (rather than a project)

4. Make it accessible for organizational use in Analysis Workspace

Once saved, these company templates are available to all users in your organization under the Templates tab.

Customizing Marketing Channel Derived Field Logic

If you want to customize the underlying marketing channel definition itself (not just the report template), you can modify the derived field logic in your Data View:

1. Navigate to your Data View in Customer Journey Analytics

2. Select the [!UICONTROL Components] tab

3. Locate your existing Marketing Channel derived field

4. Edit the logic to:

  • Modify output channel names
  • Add logic for additional channels specific to your organization
  • Adjust rule conditions to match your business requirements

5. Select [!UICONTROL Save]
