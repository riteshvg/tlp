Configuring `productListItems` in AEP Web SDK

The `productListItems` object is used within the `xdm` payload of a `sendEvent` call to capture product-related commerce data. It is structured as an array, allowing you to include multiple products in a single event.

Basic Structure

`productListItems` is passed inside the `xdm` object alongside a `commerce` event type. The general pattern is:

```javascript

alloy("sendEvent", {

"xdm": {

"commerce": {

"<commerceEventType>": {

"value": 1

}

},

"productListItems": [

{

"SKU": "PRODUCT_SKU",

"name": "Product Name",

"quantity": 1,

"priceTotal": 29.99

}

]

}

});

```

Common Commerce Event Examples

1. Product Views

Use `productViews` in the `commerce` object when a product detail page is viewed:

```javascript

alloy("sendEvent", {

"xdm": {

"commerce": {

"productViews": {

"value": 1

}

},

"productListItems": [

{

"SKU": "HT105",

"name": "The Big Floppy Hat"

},

{

"SKU": "HT104",

"name": "The Small Floppy Hat"

}

]

}

});

```

2. Add to Cart (`productListAdds`)

Use `productListAdds` when a product is added to the cart. You can also include additional fields like `productAddMethod`:

```javascript

alloy("sendEvent", {

"xdm": {

"commerce": {

"productListAdds": {

"value": 1

}

},

"productListItems": [

{

"SKU": "HT105",

"name": "The Big Floppy Hat",

"quantity": 1,

"priceTotal": 29.99,

"productAddMethod": "Add to Cart Button"

},

{

"SKU": "HT104",

"name": "The Small Floppy Hat",

"quantity": 1,

"priceTotal": 9.99,

"productAddMethod": "Add-on"

}

]

}

});

```

3. Checkout (`checkouts`)

Use `checkouts` when a user initiates or progresses through checkout:

```javascript

alloy("sendEvent", {

"xdm": {

"commerce": {

"checkouts": {

"value": 1

}

},

"productListItems": [

{

"SKU": "HT105",

"name": "The Big Floppy Hat",

"quantity": 1,

"priceTotal": 29.99

},

{

"SKU": "HT104",

"name": "The Small Floppy Hat",

"quantity": 1,

"priceTotal": 9.99

}

]

}

});

```

Key Fields in `productListItems`

| Field | Description |

|---|---|

| `SKU` | Stock Keeping Unit ŌĆö unique product identifier |

| `name` | Human-readable product name |

| `quantity` | Number of units |

| `priceTotal` | Total price for this line item |

| `productAddMethod` | Method used to add product (e.g., "Add to Cart Button") |

Key Points

  • `productListItems` is always an array, even for a single product.
  • It is always paired with a `commerce` event type (e.g., `productViews`, `productListAdds`, `checkouts`, `purchases`, etc.).
  • The data is sent in XDM format, which the Edge Network then maps to downstream Adobe solutions like Adobe Analytics and Adobe Target.
